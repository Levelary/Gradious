Files:

1. index.html :
	It connects to the javascript files and runs it.

2. card.js :
	It contains the card class and its subclasses (makePayment and displayDetails).


Instructions to run the code:

1. Download and unzip the zip file.

2. Run the "index.html" file on the web browser.

3. Inspect the page by either clicking "ctrl+shift+I" ("cmd+opt+I" in Mac) or right-click and select the "Inspect" option.

4. Open the console on the inspect tab to view the output.
